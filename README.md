# simple-gps

My personal landing page.

## Stack

- FastAPI
- Python 3.13+
- Azure App Service

## Run locally

``` bash
uv pip install -r requirements.txt
```

```bash
uv run main.py
```

Visit `http://localhost:8000`

## Deploy

Deployed via Azure App Service using `azd`:

```bash
azd up
```

Infrastructure defined in `infra/` using Bicep.
